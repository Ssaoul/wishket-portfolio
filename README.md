# wishket-portfolio

위시켓 지원용 제안 데모 모음.

## 구조

```
/                         공통 인덱스 (전체 데모 목록)
/p/<프로젝트id>-<슬러그>/   제안별 데모 상세
```

Vercel 정적 호스팅. 폴더 구조가 그대로 URL 이 된다.

| 경로 | 내용 |
|---|---|
| `/` | 공통 포트폴리오 인덱스 |
| `/p/156968-b2b-publishing/` | B2B 솔루션 기업 정적 웹 퍼블리싱 제안 데모 |
| `/p/156943-matching-platform/` | 청년-지역 매칭 플랫폼 — 공고·지원·선발 확정 원스톱 프로토타입 |

## 원칙

- **여기 있는 건 전부 제안용 데모다.** 납품한 고객 프로젝트가 아니다.
  각 페이지와 위시켓 포트폴리오 양쪽에 그 사실을 명시한다.
- 발주가 브랜드 가이드(컬러·폰트·기준 폭)를 명시했다면 임의로 해석하지 않고 그대로 지킨다.
- 발주처·클라이언트는 특정하지 않는다. 공개 페이지이기 때문.
